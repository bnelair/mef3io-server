# Contributing

Contributions are welcome. The full guidelines — development setup, pull-request
expectations, and how to report good bugs — are in
[`CONTRIBUTING.md`](https://github.com/bnelair/mef3io-server/blob/main/CONTRIBUTING.md)
at the repository root.

In short:

- **Report bugs / request features** via
  [GitHub issues](https://github.com/bnelair/mef3io-server/issues).
- **Send pull requests** from a branch off `main`; add or update tests and keep
  the suite green (`pytest`).
- **Report security issues privately** — see
  [`SECURITY.md`](https://github.com/bnelair/mef3io-server/blob/main/SECURITY.md),
  not a public issue.

By contributing you agree that your contributions are licensed under the
project's [Apache 2.0 license](https://github.com/bnelair/mef3io-server/blob/main/LICENSE).
